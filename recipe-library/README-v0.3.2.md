# Recipe Library v0.3.2 — Home

This delivery replaces the Home page with the approved editorial direction.

## Files

- `app/page.tsx`
- `app/globals.css`
- `components/HomeRecipeCard.tsx`

## Install

Copy the three folders into the repository root and allow Windows to replace
`app/page.tsx` and `app/globals.css`.

Then run:

```powershell
npm run dev
```

Review:

```text
http://localhost:3000
```

## Build check

```powershell
npm run build
```

## Commit

```powershell
git add app/page.tsx app/globals.css components/HomeRecipeCard.tsx
git commit -m "feat: redesign editorial home page"
git push origin main
```
