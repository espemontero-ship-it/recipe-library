Recipe Library v0.7.0 — Browse search and filters

Adds:
- Multi-word search across title, author, searchable ingredients and original ingredient lines.
- Multiple ingredient filters using AND logic.
- Dish, format, meal type, cooking method and cuisine filters.
- Combined filters across categories.
- Removable active-filter chips and clear-all action.
- Result count, title sorting, recently-updated sorting.
- Grid and list views.
- Recipe links use slugs.
- Collections are intentionally ignored.

Install:
1. Stop npm run dev.
2. Copy the app folder over the project app folder.
3. Confirm replacement of app/browse/page.tsx and app/browse/browse.module.css.
4. Run npm run dev.
5. Open http://localhost:3000/browse
