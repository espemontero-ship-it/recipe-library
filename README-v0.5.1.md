# Recipe Library v0.5.1 — Browse 2.0

- Adaptive cards: recipes without photos no longer show fake or empty image space.
- Instant search across title, ingredients, author, publication and tags.
- Dynamic filter chips.
- Favorites filter and favorite toggle.
- Sort by recent, alphabetical, protein or calories.
- Grid and list views.
- Responsive layout.

## Install

Copy `app/browse` into the project and replace the existing folder.

## Test

```powershell
npm run dev
```

Open `http://localhost:3000/browse`.
