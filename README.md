# Recipe Library v0.3.1

Interactive frontend foundation for Recipe Library.

## Included
- Next.js 15 + React 19 + TypeScript
- Responsive editorial UI
- Home, Browse, Recipe, Collections and Paste Recipe
- Mock recipe data

## Run locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

Suggested commit: `v0.3.1 frontend foundation`
