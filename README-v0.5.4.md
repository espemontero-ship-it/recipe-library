# Recipe Library v0.5.4

Adds Browse filters using the agreed taxonomy:

- Main ingredients
- Dish type
- Cooking method
- Cuisine
- Collections
- Source
- Author
- Tested / Not tested
- Rating
- Favorite
- This Weekend

Also separates Favorite, Tested and This Weekend so they can coexist.

Install by copying the `app` and `lib` folders over the project root.

Run:

```powershell
npm run dev
```
